VR Web Shop

Virtual Reality Webshop.

Virtual reality (VR) is a transformative technology that allows users to experience computer-generated environments in a realistic and immersive manner. It has revolutionized various industries such as gaming, healthcare, education, and retail. VR applications provide users with engaging and interactive experiences by simulating virtual environments where they can explore products, visualize them in 3D, and even interact with them. This project presents a VR e-commerce application developed using Unity and React, offering users a seamless and immersive shopping experience. The application integrates the virtual environment with a web-based front-end, allowing users to browse products and make purchases without leaving the VR environment. By leveraging the XR interaction toolkit, users can interact with objects using hand presence and switch between grab and ray interaction modes. The application also incorporates a virtual web browser using the Vuplex 3D WebView for Android asset. By combining VR technology, web development, and intuitive user interfaces, this project showcases the potential of VR in creating immersive retail experiences. As VR technology continues to advance, it holds promising prospects for integration into various aspects of our daily lives, including education, creativity, and remote collaboration.

  • Date: April 2023
  • Category: Unity VR Development/React and Springboot.
  • Github Link.

Stone TakeAway Game

A game develped using JavaFX

The following game is played by two players. Consider 15 small boxes arranged in a line. Each box except one contains a piece of stone. (Thus, there is one empty box. When a game is started a box must be chosen to be the empty one randomly.) Players move in turn. In a move, a player must take away a stone from a box or two stones from two adjacent boxes. The winner of the game is the player who makes the last move. When the game is started the program asks for the names of the players. The program stored the result of the games as follows. For each game, the following information is stored: the date and time when the game was started, the name of the players, the number of moves made by the players during the game, and the name of the winner. The program displays a high score table in which the top 5 players with the most wins are displayed. The data is stored in a JSON file.

  • Date: May 2022
  • Category: JavaFX Development
  • Github Link.
